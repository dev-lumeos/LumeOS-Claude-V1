# LumeOS Obsidian Vault

Human-facing interface für LumeOS Dokumentation und Memory.

## Struktur

- **01-Architecture**: System Architektur Dokumentation
  - Source: `docs/architecture/`
- **02-Decisions**: Architecture Decision Records (ADRs)
  - Source: `docs/decisions/`
- **03-WO-Batches**: Work Order Batches
  - Source: `system/workorders/batches/`
- **04-Memory**: Canonical Memory
  - Source: `system/memory/canonical/`
- **05-Benchmarks**: Benchmark Reports
  - Source: `docs/reports/`

## Nutzung

1. Öffne diesen Ordner als Vault in Obsidian
2. Navigiere via Sidebar zu den verschiedenen Bereichen
3. Nutze Obsidian Search für schnelle Navigation
4. Graph View zeigt Verlinkungen zwischen Dokumenten

## Sync

Die Inhalte werden aus den Source-Verzeichnissen gespiegelt.
Änderungen sollten in den Source-Verzeichnissen gemacht werden.
