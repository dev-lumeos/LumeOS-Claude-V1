# ADR Index

Architecture Decision Records für LumeOS.

## Index

| ID | Date | Topic | Status |
|----|------|-------|--------|
| - | - | Keine ADRs bisher | - |

## Konventionen

- Format: `ADR-{id}-{date}-{topic}.md`
- ID: Fortlaufende Nummer (001, 002, ...)
- Date: ISO Format (YYYY-MM-DD)
- Topic: Kurze Beschreibung in kebab-case

## ADR Template

```markdown
# ADR-{id}: {Title}

## Status
Proposed | Accepted | Deprecated | Superseded

## Context
Was ist das Problem oder die Situation?

## Decision
Was haben wir entschieden?

## Consequences
Was sind die Auswirkungen dieser Entscheidung?

## Alternatives Considered
Welche Alternativen wurden betrachtet?
```
